Proyectos ()

Intranet Hospital Sopo

Intranet para el hospital Divino Salvador de Sopó. Se realiza en WordPress y con manipulación del código en PHP se logra dar funcionalidad y cubrir los requerimientos solicitados por el hospital Divino Salvador de Sopó. El proyecto se monto en el servidor hosting en la nube.

Consentimientos

Programa realizado en Python para escaneo de consentimientos informados de los pacientes del hospital. Su función es verificar si el documento existe y organizarlo en la carpeta correspondiente para posteriormente subirlo al sistema de información CitiSalud.

Hoja vida PC

Software desarrollado en Python que a partir de un archivo .csv de las características de computador genera una hoja de vida en Excel del equipo a examinar, esto para mejorar el proceso de la creación de las hojas de vida de los equipos del Hospital Divino Salvador de Sopó.

Whatsapp automaticos

Programa desarrollado en Python, se ejecuta en una ventana de comandos de Windows y su función es tomar números de una hoja de Excel y enviar mensajes a través del WhatsApp web. Se utiliza para recordar citas a los usuarios o programas de demanda inducida.
